# AS-Cli

#### 介绍

AS-Cli 是一个脚手架工具项目

#### 使用说明

1.  npm i as-yyds-cli -g (从 npm 全局安装 AS-Cli)
1.  as -h (查看脚手架所有的指令)
1.  as create name (创建项目)

#### 本地调试

1.  npm install (安装依赖)
2.  npm link （进行全局 link 注册）
3.  npm unlink （卸载全局 link）

#### 使用的 npm 库

```
commander                      // 快速开发Nodejs命令行工具的package
inquirer                       // 给输入命令行的用户提供一个好看的界面
download-git-repo              // 从git下载项目
```
